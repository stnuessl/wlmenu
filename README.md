# wlmenu

A simple and fast application launcher for wayland.

## Motivation

## Installation

### Dependencies

* wayland 1.16
* cairo 1.16
* libxkbcommon-x11 0.8.3
* freetype2 2.9.1

### Compilation

    $ git clone https://github.com/stnuessl/wlmenu
    $ cd wlmenu
    $ make CC=gcc

## Usage

    $ wlmenu --help
